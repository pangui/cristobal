---
name: Numerar los supuestos
description: Cuando liste supuestos para que CCS los confirme/corrija, usar numeración (1, 2, 3…), no bullets.
type: feedback
created: 2026-04-25
---

Cuando enumere supuestos que tomo para que CCS los confirme o corrija, **uso numeración (1., 2., 3.)**, no bullets.

**Why:** CCS lo pidió el 2026-04-25 después de que enlisté supuestos con bullets. Los números son referenciables ("el 3 está mal", "ok el 1 y 2"). Los bullets obligan a copiar el texto del supuesto para identificarlo.

**How to apply:**
- Aplica a listas de supuestos, premisas, o ítems destinados a aprobación discreta.
- No aplica a listas descriptivas (componentes, archivos, pasos secuenciales internos) — ahí los bullets siguen bien.
- Mantengo cada supuesto en una línea o un párrafo corto, para que el número sea suficiente para señalarlo.
